# Family Memory Center — 定位与架构

> 版本：1.0 | 日期：2026-04-09

---

## 一、核心定位

### 1.1 是什么

Family Memory Center（记忆中心）是家庭智能体系统的**中央知识库**，负责：

- 存储和管理家庭成员的所有知识
- 为各 Agent 提供记忆查询服务
- 协调多成员记忆的共享与隔离
- 管理知识的生命周期（创建→衰减→归档）

### 1.2 服务对象

| 对象 | 权限 |
|------|------|
| **主 Agent** | 读写所有家庭共享记忆、成员共享记忆 |
| **领域 Agent** | 按需读取相关记忆、写入专属领域记忆 |
| **用户（直接查询）** | 只读视角，可查看但不能直接修改 |
| **成员 Agent** | 读写自己成员的私有记忆 |

### 1.3 核心规则

```
┌─────────────────────────────────────────────────────┐
│                    记忆中心规则                      │
├─────────────────────────────────────────────────────┤
│ 1. 家庭共享记忆：所有 Agent 可读写                   │
│ 2. 成员共享记忆：家庭成员 + 主 Agent 可读写          │
│ 3. 成员私有记忆：仅对应成员 + 主 Agent 可读写        │
│ 4. 用户查询：只读，按规则过滤敏感内容               │
│ 5. 写入必须通过渐进学习机制（被动/主动确认）         │
│ 6. 删除必须通过用户确认                              │
└─────────────────────────────────────────────────────┘
```

## 二、系统架构

### 2.1 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                        外部调用层                             │
├─────────────────────────────────────────────────────────────┤
│  主Agent │ 领域Agent │ 用户查询接口 │ 备份恢复接口            │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│                       查询引擎层                             │
├─────────────────────────────────────────────────────────────┤
│  自然语言查询 → 意图分类 → RAG检索 → 结果过滤 → 返回          │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│                       存储引擎层                             │
├─────────────────────────────────────────────────────────────┤
│         SQLite（结构化数据）│ 向量索引（语义搜索）            │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│                       知识管理层                             │
├─────────────────────────────────────────────────────────────┤
│  知识节点 │ 版本管理 │ 生命周期管理 │ 冲突检测                │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 核心组件

| 组件 | 职责 |
|------|------|
| **Query Engine** | 处理自然语言查询，路由到不同检索策略 |
| **RAG Retriever** | 语义向量检索，找到最相关的记忆 |
| **Knowledge Graph** | 知识关系图谱，管理实体之间的关系 |
| **Access Controller** | 访问控制，按规则过滤结果 |
| **Lifecycle Manager** | 知识生命周期（创建→活跃→衰减→归档） |
| **Backup Manager** | 备份与恢复管理 |

## 三、记忆分类（按可见性）

### 3.1 三层记忆

| 层级 | 说明 | 可访问者 |
|------|------|---------|
| **家庭共享** | 所有成员共同的知识 | 全体成员 + 全体 Agent |
| **成员共享** | 同属一个成员的知识 | 该成员 + 主 Agent |
| **成员私有** | 极度私密的个人记忆 | 仅该成员（或指定代理人） |

### 3.2 家庭共享记忆

| 类型 | 示例 |
|------|------|
| 家庭规则 | "家里不准在餐厅吃外卖" |
| 共同习惯 | "每周五晚上是家庭电影夜" |
| 家庭日程 | "每月第一个周六家庭聚会" |
| 共享资源 | "小区停车位编号是 B123" |
| 家庭偏好 | "全家人都不吃香菜" |

### 3.3 成员共享记忆

| 类型 | 示例 |
|------|------|
| 健康信息 | "爸爸有高血压" |
| 工作信息 | "妈妈在字节工作" |
| 偏好信息 | "爸爸喜欢吃川菜" |
| 习惯信息 | "爸爸每周跑步3次" |

### 3.4 成员私有记忆

| 类型 | 示例 |
|------|------|
| 私人想法 | "儿子暗恋同班同学" |
| 个人秘密 | "妈妈藏了一些私房钱" |
| 情感记录 | "女儿和朋友吵架了" |

## 四、访问控制规则

### 4.1 读取规则

```
查询者发起查询
       │
       ▼
┌──────────────────┐
│ 确定查询者身份     │
└──────────────────┘
       │
       ▼
┌──────────────────────────────────┐
│ 过滤记忆列表                      │
├──────────────────────────────────┤
│ if 家庭共享 → 保留                │
│ if 成员共享 → 查询者是该成员？保留  │
│ if 成员私有 → 查询者是该成员或主Agent？保留 │
│ else → 移除                       │
└──────────────────────────────────┘
```

### 4.2 写入规则

```
写入请求
       │
       ▼
┌──────────────────────────────────┐
│ 来源检测                          │
├──────────────────────────────────┤
│ if 来源=渐进学习机制 → 正常处理   │
│ if 来源=用户直接 → 转为待确认     │
│ if 来源=其他Agent → 权限检查      │
└──────────────────────────────────┘
```

### 4.3 用户直接查询规则

| 查询场景 | 返回内容 |
|---------|---------|
| 问"家里有什么习惯？" | 返回家庭共享记忆 |
| 问"我的日程安排" | 返回自己成员的共享记忆 |
| 问"其他成员在忙什么？" | 仅返回共享日程，不含私有 |
| 问"老公有什么秘密？" | 拒绝，返回"无权访问" |

## 五、与各 Agent 的交互协议

### 5.1 查询协议

```json
{
  "action": "query",
  "query": "今天有什么安排",
  "requester": "life-agent",
  "scope": ["family_shared", "member_shared"],
  "member_id": null,
  "filters": {
    "time_range": "today",
    "types": ["schedule", "reminder"]
  }
}
```

### 5.2 写入协议

```json
{
  "action": "write",
  "knowledge": {
    "type": "habit",
    "content": "每周五晚上叫外卖",
    "confidence": 0.8
  },
  "source": "learning-mechanism",
  "auto_confirm": false
}
```

### 5.3 删除协议

```json
{
  "action": "delete",
  "knowledge_id": "xxx",
  "reason": "user_confirmed",
  "requester": "user"
}
```
