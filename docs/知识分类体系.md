# 知识分类体系

> 四层分类：事实 / 偏好 / 习惯 / 禁忌

---

## 一、事实（Facts）

### 定义
客观可验证的信息，不会随个人意愿改变。

### 特征
- 可以用"真假"判断
- 长期有效，不会过期
- 通常一次学习即可确认

### 示例

| 类型 | 示例 |
|------|------|
| 家庭成员 | "我爸叫张三" |
| 地理位置 | "我们小区在朝阳区" |
| 日期事件 | "我生日是6月15日" |
| 职业信息 | "我在字节工作" |
| 设备状态 | "空调品牌是小米" |

### 学习优先级
- **高**：成员信息、住址、紧急联系方式
- **中**：工作信息、常用设施
- **低**：设备型号等可选项

---

## 二、偏好（Preferences）

### 定义
个人主观选择，没有对错之分。

### 特征
- 可以改变，不需要验证
- 通常影响推荐和排序
- 需要多次确认才能稳定

### 示例

| 类型 | 示例 |
|------|------|
| 饮食偏好 | "我喜欢吃川菜" |
| 出行偏好 | "我更喜欢坐高铁而不是飞机" |
| 购物偏好 | "我一般用京东买东西" |
| 娱乐偏好 | "周末我喜欢看电影" |
| 信息偏好 | "我希望早上看新闻" |

### 学习优先级
- **高**：饮食偏好、健康相关偏好
- **中**：娱乐、购物偏好
- **低**：信息获取习惯

### 置信度衰减
- 偏好如果3个月未触发，置信度降低50%

---

## 三、习惯（Habits）

### 定义
规律性行为模式，有时间/频率特征。

### 特征
- 通常有时间维度（每周X、每天Y）
- 可以预测未来行为
- 需要多次观察才能确认

### 示例

| 类型 | 示例 |
|------|------|
| 作息习惯 | "我一般11点睡觉" |
| 餐饮习惯 | "每周五晚上我叫外卖" |
| 运动习惯 | "我每周末跑一次步" |
| 购物习惯 | "我每月15号网上购物" |
| 社交习惯 | "我每周六晚上和朋友聚会" |

### 学习优先级
- **高**：家庭共同习惯
- **中**：个人核心习惯（影响健康/财务）
- **低**：低频习惯（偶发行为不计入）

### 置信度衰减
- 习惯如果2个月行为不一致，标记"可能已变"

---

## 四、禁忌（Taboos）

### 定义
绝对不能违反的底线。

### 特征
- 违反会造成严重后果
- 通常一次学习即可
- 不随时间改变

### 示例

| 类型 | 示例 |
|------|------|
| 饮食禁忌 | "我对海鲜过敏" |
| 语言禁忌 | "不要在我面前提某某话题" |
| 行为禁忌 | "不要在我睡觉时开灯" |
| 信息禁忌 | "不要告诉任何人我的收入" |
| 商业禁忌 | "不要推荐任何P2P投资" |

### 学习优先级
- **最高**：健康相关禁忌（过敏、急救）
- **高**：隐私相关禁忌
- **中**：行为习惯禁忌

### 置信度衰减
- **永不衰减**，除非用户主动修改

---

## 五、四层对比

| 维度 | 事实 | 偏好 | 习惯 | 禁忌 |
|------|------|------|------|------|
| 可验证性 | 高 | 低 | 中 | 高 |
| 变化频率 | 低 | 高 | 中 | 极低 |
| 学习难度 | 易 | 中 | 难 | 易 |
| 遗忘风险 | 低 | 中 | 高 | 极低 |
| 置信度衰减 | 无 | 3个月 | 2个月 | 永不 |
| 影响程度 | 参考 | 推荐 | 预测 | 强制 |
