## 三、详细机制

### 3.1 知识分类体系

详见：`知识分类体系.md`

| 类型 | 衰减规则 | 置信度 |
|------|---------|-------|
| 事实 | 6个月-10% | 高 |
| 偏好 | 3个月-50% | 中 |
| 习惯 | 2个月不一致→待确认 | 低→中 |
| 禁忌 | 永不衰减 | 永久 |

### 3.2 被动学习机制

详见：`被动学习机制.md`

**触发条件：** 对话结束自动分析

**处理流程：**
1. 扫描可学习信号
2. 识别知识类型
3. 计算置信度
4. 检测冲突
5. 写入知识库

**限制：** 单次对话最多学习10条

### 3.3 主动确认机制

详见：`主动确认机制.md`

**触发条件：**
- 健康/财务/关系/禁忌类信息
- 覆盖旧知识
- 置信度 < 60%

**确认方式：**
- 即时确认（高重要性）
- 延迟确认（低重要性，积累后确认）
- 批量确认（定期汇总）

### 3.4 冲突处理机制

详见：`知识冲突处理.md`

**核心原则：** 观察3次，少数服从多数

**处理流程：**
```
冲突检测 → 观察模式 → 记录3次 → 决策（替换/保留/并存）
```

**决策规则：**
- 3次中≥2次支持新知识 → 新替旧
- ≤1次支持新知识 → 保留旧
- 支持率相同 → 保留两者

### 3.5 衰减机制

详见：`知识衰减机制.md`

**衰减周期：**

| 类型 | 触发衰减 | 标记待确认 | 降级/归档 |
|------|---------|----------|---------|
| 事实 | 6个月 | 18个月 | 24个月 |
| 偏好 | 3个月 | 6个月 | 12个月 |
| 习惯 | 2个月不一致 | 3个月 | 6个月 |
| 禁忌 | 永不 | 永不 | 永不 |

**待确认处理：**
- 仍可使用，置信度降低
- 用户主动提到 → 自动解除
- 月度清理时征询用户
