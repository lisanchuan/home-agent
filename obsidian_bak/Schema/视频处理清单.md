# 视频笔记处理清单

每次处理视频时，对照检查：

## 必做项

- [ ] 确认视频平台（B站/抖音/YouTube）
- [ ] 调用 BibiGPT 分析视频
- [ ] 保存到 Raw 层（原始笔记）
- [ ] 提炼核心内容保存到 Wiki 层
- [ ] 建立 Raw ↔ Wiki 双向链接

## 内容检查

- [ ] 标题是否准确
- [ ] 摘要是否覆盖核心观点
- [ ] 亮点是否标注时间戳
- [ ] 思考是否有价值
- [ ] 术语是否解释清楚
- [ ] 标签是否准确

## 产出检查

- [ ] 是否适合存 Raw（原始素材）
- [ ] 是否适合存 Wiki（已提炼知识）
- [ ] 是否适合存 Outputs（创作产物）
- [ ] 是否需要更新汇总

## 标签规范

| 类型 | 标签格式 |
|------|---------|
| 原始素材 | #raw #video #平台 |
| 知识提炼 | #知识管理 #AI #... |
| 创作产出 | #汇总 #文章 #报告 |

---

*使用说明：处理视频前打印或复制此清单，对照执行*
