# 知识库结构定义

## 三层架构

```
知识库/
├── Raw/          # 原始素材层（只读）
├── Wiki/         # AI 维护的结构化知识层
├── Schema/       # 协作协议定义
└── Outputs/      # 创作输出
```

## 目录说明

| 目录 | 用途 | AI 权限 |
|------|------|---------|
| Raw | 原始资料（文章、截图、笔记） | 只读 |
| Wiki | AI 提炼的结构化知识 | 读写 |
| Schema | 规则定义和系统配置 | 只读 |
| Outputs | 创作产物和最终输出 | 读写 |

## 分类规则

- 每篇文章按主题分类到对应 Wiki 文件夹
- 使用双向链接 `[[page]]` 连接相关内容
- 定期运行 Lint 健康检查

## 标签系统

- #知识管理 #AI #学习方法 #项目 #待整理
- 视频笔记标签：#视频笔记 #bilibili
- 标签格式：使用小写字母和连字符

## 处理规范

- 视频笔记处理参照 [[Schema/视频处理清单]]
- 创作产出存入 [[Outputs]] 目录
- Raw → Wiki → Outputs 的流动路径
