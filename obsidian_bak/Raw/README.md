# Raw 原始素材层

此文件夹存放所有未经处理的原始资料。

## 规则

- **AI 权限**：只读
- **存入内容**：文章、网页截图、PDF、笔记草稿
- **命名规范**：`日期-来源-标题`
- **处理流程**：存入后调用 AI ingest 提炼到 Wiki

## 示例

```
Raw/
├── 2026-04-10-bilibili-Karpathy知识库视频1.md
├── 2026-04-10-article-AI发展趋势.md
└── 2026-04-10-notes-读书笔记.md
```
