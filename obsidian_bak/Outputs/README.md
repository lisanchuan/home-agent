# Outputs 创作输出

你的创作产物从这里产出。

## 规则

- **AI 权限**：读写
- **内容类型**：汇总、文章、博客、方案、报告
- **命名规范**：`日期-类型-标题`
- **来源**：从 Wiki 层提炼的创作，不是原始素材

## 目录说明

- 汇总：多篇内容整合（如系列汇总）
- 文章：独立创作的深度内容
- 报告：分析报告、调研报告

## 示例

```
Outputs/
├── 2026-04-10-汇总-Karpathy知识库系列演进.md
├── 2026-04-11-文章-AI工具选型指南.md
└── 2026-04-12-报告-本周学习总结.md
```
